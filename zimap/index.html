Zi9's Custom Map Loader for CarX

Loader Plugin

This is the plugin that enables CarX to recognize and load custom maps.

Custom maps will appear in all default map selection menus.

Quickstart guide:

  1. Install BepInEx Mod Loader
  2. Install the Map Loader plugin
  3. Install some maps
  4. Enjoy the custom map experience

Map Index

Here you can find all community created and ported maps.

To install a map, download it and move it into the ZiMaps folder.
