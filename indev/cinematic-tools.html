Cinematic Tools

"Say no to shaky mouse cinematics"

Cinematic Tools is a plugin for CarX that allows the player to record smooth camera animations.

This plugin will allow you to set start and end points for your camera and allow you to set the time it will animate the camera between those points.

It also features animatable FOV and Tilt controls.

Installation is described in the included Readme.txt file in the zip.
