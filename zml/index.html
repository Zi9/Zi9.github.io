Welcome to

ZML

ZML is a new modification loader for CarX Drift Racing Online.

It is built as a new, more secure platform for modding the game.


Features

  • 🔒

    Mod encryption

  • 🔑

    Private mod support (SteamID based locking)

  • 🖇️

    BepInEx mod support (only approved mods)

  • 🖼️

    A central UI for mods to use (ZML Manager Toolbox)

  • ⌨️

    A keybind API for mods to easily define re-bindable keys

  • ⚙️

    A config API for mods to easily set and get settings

  • ⚔️

    Built-in anti-cheat & anti-grief

  • 🔄

    Automatic updating

  • ➕

    ... and much more
