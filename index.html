Cinematic Tools

"Say no to shaky mouse cinematics"

Cinematic Tools is a plugin for CarX that allows the player to record smooth camera animations.This plugin will allow you to set start and end points for your camera and allow you to set the time it will animate the camera between those points.It also features animatable FOV and Tilt controls.Installation is described in the included Readme.txt file in the zip.

Community Sticker Pack

This pack contains 1231 new stickers. It has a built in update detection mechanism that will notify you when the pack is updated. Stickers that are white color are colorable. This pack works similiarly to car mods, you need to have the pack installed to see other people with custom stickers online. To remove the pack be sure to use the provided uninstaller in the CarX folder and follow instructions.

Big thanks to Huskyy for a lot of help during development.
