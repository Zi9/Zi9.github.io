Cinematic Tools

"Say no to shaky mouse cinematics"

Cinematic Tools is a plugin for CarX that allows the player to record smooth camera animations.This plugin will allow you to set start and end points for your camera and allow you to set the time it will animate the camera between those points.It also features animatable FOV and Tilt controls.Installation is described in the included Readme.txt file in the zip.

Community Sticker Pack

This pack contains 1231 new stickers. It has a built in update detection mechanism that will notify you when the pack is updated. Stickers that are white color are colorable. This pack works similiarly to car mods, you need to have the pack installed to see other people with custom stickers online. To remove the pack be sure to use the provided uninstaller in the CarX folder and follow instructions.

Big thanks to Huskyy for a lot of help during development.

Assetto Corsa Soundbank Loader Plugin

This plugin allows you to use soundbanks from the game Assetto Corsa and replace the default ingame sounds with the ones from the soundbank. It uses the FMOD sound engine.

Sadly it has been broken due to CarX implementing FMOD themselves and that breaks my implementation. Additionally it was going to have a menu where you can select which soundbank to load and automatic loading of the bank you select on race start. Currently development of this plugin has been put on hold but there is a chance i will revisit it sometime in the future.

Custom Map Loader Plugin

This plugin allows you to load a custom map from an OBJ format 3D model and use it as a track in the game. To make collisions work correctly you need to have a single mesh OBJ for collisions. This plugin also doesn't touch the original track's collision model so you can just go back to the original track without even leaving the lobby. Development has been put on hold. Planned features were going to inclue a menu where you select which map to load and the ability to disable and enable maps ingame.

Big thanks to Huskyy and Mr.CoolGoat for help during development
